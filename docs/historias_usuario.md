# Historias de usuario
## Historia 1: Registrar Usuario
Como usuario vecino, quiero registrarme para poder iniciar sesión y luego registrar las incidencias
Criterios: Debo ingresar Nombre de usuario, correo, contraseña

## Historia 2: Iniciar Sesión
Como usuario vecino, quiero iniciar sesión para poder acceder al formulario de registro de incidia
Criterios: Debo ingresar Correo, contraseña

## Historia 3: Registrar incidencia
Como usuario vecino, quiero registrar un problema en mi calle para que la municipalidad pueda revisarlo.
Criterios:
- Debo ingresar título, descripción, tipo y dirección.
- Puedo adjuntar también una evidencia (en ese caso foto)

## Historia 4: Visualizar cambios
Como trabajador municipal, quiero ver el registro de incidencias.

Criterios:
- Las incidencias deben permanecer en la base de datos
- Se debe contar con un solo botón "actualizar" para revisar las evidencias.
